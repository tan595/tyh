# HGGEP

- 对应论文：`/root/workspace/1/论文/HGGEP.pdf`
- 对应代码：`/root/workspace/1/HGGEP-main`
- 一句话定位：用梯度增强、多层特征注意力和超图网络，从病理图像中预测空间基因表达。

## 1. 摘要
- 背景：ST 很有价值，但成本高；从组织学图像预测表达是更可扩展的替代思路。
- 挑战：现有方法往往忽略细胞形态梯度信息，也没有充分利用不同深度层次的图像特征。
- 方法：HGGEP 引入 gradient enhancement module 强化细胞形态边界与纹理，用多层特征抽取和注意力模块处理不同潜层表示，再用超图建模多尺度高阶关联。
- 贡献：在 HER2+ 和 cSCC 等数据上，相比既有方法取得更高 PCC，并在 SSIM/RMSE 等指标上也更优。

## 2. 引言
- 问题背景：图像里不仅有局部纹理，还有跨区域结构关系，这些信息都和基因表达有关。
- 历史发展：ST-Net 开启了图像到表达的 CNN 路线；HisToGene、Hist2ST 引入 Transformer/GNN；THItoGene 和 BLEEP 分别强化了结构建模和对比学习。
- 作者动机：作者认为已有方法还是不够关注细胞形态学细节，同时把不同层次特征割裂使用，没有建立跨尺度高阶关系。
- 方法与贡献落点：HGGEP 试图把“局部梯度细节”“多层潜在特征”“跨尺度高阶关系”统一起来。

## 3. 相关工作
- 问题发展历史：图像预测表达的主线从 CNN 回归发展到 Transformer、图网络和对比学习三类增强思路。
- 算法发展历史：普通图网络适合点对点或边对边关系，但对多尺度特征之间的高阶互动建模不够自然。
- 本文相对差异：HGGEP 把超图引入该任务，本质上是在问“多个尺度特征能否作为一个 group 一起参与建模”，这比普通图更强。

## 4. 方法
- 动机：真实组织中的基因表达既受细胞本身形态影响，也受周围微结构和多尺度背景影响，因此需要层次化建模。
- 核心流程：图像先经过 GEM 强化梯度；轻量 backbone 提取多层特征；CBAM/ViT 对每层特征做表示优化；随后 HAM 超图模块建模跨层高阶关系，最后接 LSTM 与回归头输出表达。
- 算法理解：GEM 负责“看清边界”，注意力模块负责“挑出有效特征”，HAM 负责“把不同尺度的信息组合成更高阶的表达线索”。
- 关键收益：这套结构特别适合病理图像，因为病理信息往往天然就是多尺度叠加的。

## 5. 实验
- 实验内容：在 HER2+ 与 cSCC 数据上做留一验证，对比 Hist2ST、HisToGene、ST-Net，并用 PCC、SSIM、RMSE 和消融实验评估。
- 实验现象：HGGEP 在平均 PCC 和中位 PCC 上都优于对比方法，在困难切片上优势更明显。
- 原因分析：梯度增强提高了细胞形态可分性，超图提升了跨层信息整合能力，因此模型在复杂区域更不容易失真。

## 6. 结论
- 总结全文：HGGEP 的核心贡献是把病理图像中的多尺度高阶结构关系显式纳入表达预测模型。
- 展望：后续可以把超图结构从手工构建升级为可学习结构，并和更强病理基础编码器结合。
