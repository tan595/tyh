# cNMF

- 对应论文：`/root/workspace/1/论文/cNMF.pdf`
- 对应代码：`/root/workspace/1/cNMF-main`
- 一句话定位：通过多次 NMF + 共识聚类，稳定地把单细胞表达分解成“细胞类型程序”和“细胞活动程序”。

## 1. 摘要
- 背景：单细胞表达是多个基因程序叠加的结果，既有细胞类型 identity program，也有 cell cycle、hypoxia 等 activity program。
- 挑战：单个细胞往往同时表达多种程序，而常规聚类或单次矩阵分解难以稳健拆开这些混合信号。
- 方法：作者提出 consensus NMF，先多次运行 NMF，再对各次得到的 program 做聚类与共识整合，得到更稳定的 gene expression programs 和 usage。
- 贡献：在模拟数据上稳定区分 identity/activity programs，并在脑类器官与视觉皮层数据中发现已知和新的活动程序。

## 2. 引言
- 问题背景：scRNA-seq 能看见细胞间差异，但每个细胞的表达常是多个程序叠加后的结果。
- 历史发展：PCA、ICA、NMF、LDA 等分解方法都被用于 scRNA 预处理，但很少系统比较其是否能恢复有生物学意义的程序。
- 作者动机：作者希望找到一种既能表达“一个细胞可同时使用多个程序”，又能避免矩阵分解随机性过大的方法。
- 方法与贡献落点：cNMF 的关键不是发明新的基本分解器，而是通过共识整合提升 NMF 的可重复性和生物学可解释性。

## 3. 相关工作
- 问题发展历史：早期 scRNA 分析更关注聚类和轨迹；随着 activity program 的重要性上升，矩阵分解开始重新受到重视。
- 算法发展历史：PCA/ICA 可得到线性成分，但可解释性一般；NMF 天然非负，更适合表达数据；问题在于不同随机初始化下结果不稳定。
- 本文相对差异：cNMF 通过“重复分解 + 共识聚类”把这种不稳定性压低，使 NMF 真正变成可用的 GEP 发现工具。

## 4. 方法
- 动机：如果一次 NMF 结果会随初始化波动，那么直接解释单次输出风险很大。
- 核心流程：对同一数据多次运行 NMF；把每次得到的 programs 聚到一起；对稳定 cluster 求共识程序；再估计每个细胞对这些程序的 usage。
- 算法理解：cNMF 的本质像是“bootstrap 版 NMF”，它关心的不是某一次最优分解，而是跨多次运行反复出现的稳定模式。
- 实践意义：这样得到的 program 更稳，也更适合下游做细胞状态解释和双细胞检测。

## 5. 实验
- 实验内容：作者在不同噪声水平的模拟数据上比较 PCA、ICA、LDA、NMF 等方法，并在类器官和视觉皮层数据上做真实验证。
- 实验现象：cNMF 对 identity 与 activity programs 的恢复更准、更稳，还能识别细胞周期、缺氧以及潜在神经分泌相关程序。
- 原因分析：共识步骤过滤掉了随机初始化带来的偶然成分，只保留反复出现的结构，因此鲁棒性更强。

## 6. 结论
- 总结全文：cNMF 的贡献在于把“可解释的非负分解”真正做成了稳定可用的单细胞程序发现工具。
- 展望：这一思想可以继续扩展到空间转录组、时间序列单细胞和多组学分解中。
