# DIALOGUE

- 对应论文：`/root/workspace/1/论文/DIALOGUE.pdf`
- 对应代码：`/root/workspace/1/DIALOGUE-master`
- 一句话定位：从空间转录组或单细胞转录组中挖掘“跨细胞类型协同变化”的 multicellular programs。

## 1. 摘要
- 背景：组织功能不是单个细胞类型独立完成的，而是多个细胞群体协同作用的结果。
- 挑战：现有方法多分析细胞内程序、细胞邻域或受体配体对，却缺少系统发现“跨细胞类型协同程序”的工具。
- 方法：DIALOGUE 定义 multicellular programs, MCPs，并通过稀疏的跨细胞类型相关分解和多层线性模型，从 spatial 或 scRNA-seq 数据中恢复这些程序。
- 贡献：在脑组织、肺癌、溃疡性结肠炎、阿尔茨海默病和免疫治疗场景中发现了具有生物学和临床意义的 MCP。

## 2. 引言
- 问题背景：单细胞技术和空间技术让我们可以观察不同细胞类型在组织中的表达状态，但“组织级协调”仍难以直接恢复。
- 历史发展：以 cNMF、LIGER 为代表的方法偏重细胞内程序；受体配体和邻域分析偏重局部交互；两者都没有显式建模多细胞协同程序。
- 作者动机：作者希望把研究对象从“单个细胞的状态”升级为“多个细胞类型共同构成的功能单元”。
- 方法与贡献落点：DIALOGUE 把来自同一微环境或同一样本的不同细胞类型看作同一实体的多视角表示，再从这些视角之间找共变结构。

## 3. 相关工作
- 问题发展历史：单细胞分析先解决聚类、轨迹和基因程序发现，之后才逐步延伸到细胞互作和空间组织结构。
- 算法发展历史：NMF/cNMF/LIGER 强调细胞内基因程序；HMRF、空间邻域方法强调组织分区；受体配体方法强调已知通路驱动的互动。
- 本文相对差异：DIALOGUE 不依赖先验通路，也不局限于邻接关系，而是通过跨细胞类型的统计共变来定义 MCP。

## 4. 方法
- 动机：如果多个细胞类型处于同一微环境，它们会同时表达一组相互协调的程序；这种协调关系应能从跨细胞类型相关性中恢复。
- 核心流程：先用 penalized matrix decomposition 找到不同细胞类型之间相关的稀疏潜变量；再用 multilevel model 回到基因层面，识别每个 MCP 对应的基因集合。
- 算法理解：DIALOGUE 的本质是“先在低维空间发现跨类型共变轴，再把这些轴解释回基因程序”，因此兼顾统计稳定性与生物可解释性。
- 结果形式：每个 MCP 由多个 cell-type-specific 程序构成，可以用来预测细胞所处环境、疾病状态或治疗反应。

## 5. 实验
- 实验内容：作者在小鼠脑的多种空间数据集、肺癌空间数据，以及跨患者的 scRNA-seq 数据上评估 DIALOGUE。
- 实验现象：DIALOGUE 能在未见测试集上恢复空间属性，并发现与动物行为、免疫激活、组织重塑、AD 衰老及免疫治疗耐药相关的 MCP。
- 原因分析：相比只看单个细胞或局部邻域的做法，MCP 把“多细胞协同”作为基本对象，因此更容易提取组织级机制信号。

## 6. 结论
- 总结全文：DIALOGUE 把单细胞/空间转录组分析从“细胞中心”推进到“组织协调中心”，是本文最大的理论和方法贡献。
- 展望：未来可以与更高分辨率空间组学、时间序列和多模态单细胞数据结合，进一步研究 MCP 的因果驱动机制。
